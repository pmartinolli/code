Comment ça marche ?

Un code JavaScript exécuté dans le navigateur de l'utilisateur interroge Wikidata et récupère les données des QIDs demandés. Il les présente dans un tableau avec des filtres basés sur une liste d'autorité de certains Main subjects (P921).

Ce programme a été vibe coded avec Claude Opus 4.1 d'abord puis avec ChatGPT 4.x en août 2025. Les données de Wikidata ont été améliorées pour répondre aux besoins du programme, principalement avec P921.

Curation

Le tableau des revues existait sous la forme d'un document PDF mis à jour annuellement sur le guide histoire des Bibliothèques UdeM. Il a été conçu par Marie-Ève Ménard et amélioré par Émilie Dalpé.

Dans ce projet, le bibliothécaire sélectionne les QIDs des revues, puis il dresse une liste des sujets principaux pour créer des boutons de filtres, puis il choisit les données à afficher dans le tableau. Pour que le projet fonctionne, il indexe les éléments Wikidata des revues. Ensuite, il mets ces éléments dans sa liste de surveillance (pour surveiller les contributions futures et le vandalisme).
